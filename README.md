# Quick start

Create Python virtualenv:

```bash
$ python3 -m venv .venv
```

Activate virtualenv:

```bash
$ source .venv\scripts\activate
```

Install dependencies ( only the on the first run):

```bash
$ pip install mkdocs
```

Run MKDocs:

```bash
$ mkdocs serve -a localhost:8001
```
