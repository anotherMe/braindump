
# diskpart

Diskpart is available on Windows versions: XP, Vista, 7, 8, 8.1 and 10.

The utility allows you to:

  * Partion
  * Remove partitions
  * Remove formatting
  * Assign and remove drive letters and mount points
  * Convert disks from basic to dynamic
  * Create and extend volumes
