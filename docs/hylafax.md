====== Hylafax ======

===== Useful commands =====

<code>
faxstat (shows you the status of HylaFAX)
faxstat -s (shows you the send status)
faxstat -r (shows received faxes)
faxalter -a now jobid (forces send retry now)
faxrm jobid (deletes fax from sendqueue)
</code>
